# Algorithms

## Project Description

C Functions for Search: Linear Search, Binary Search and Sorting: Bubble Sort, Selection Sort, Merge Sort

## How to Run

complile the program: ```gcc algorithms.c -o algorithms```

run the program: ```.\algorithms```
